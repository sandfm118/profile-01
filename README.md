# User Profile with GSAP and Video

This project creates a user profile using:

* GSAP for animations
* CSS for styling
* HTML for structure
* JavaScript for interactivity

**How to Use:**

1. Clone the repository using `git clone <remote_url>`.
2. Open the `index.html` file in your browser.
